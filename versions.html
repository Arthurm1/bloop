Pre-release versions

Every change merged in Bloop is made available in a pre-release version third parties can depend on. To know the latest available release, you can consult the release table in the Installation section. Pre-release versions provide no guarantees with regards to correctness and performance and only intend to make available those release-worthy changes and provide a playground for mantainers and early adopters to try future changes out.
